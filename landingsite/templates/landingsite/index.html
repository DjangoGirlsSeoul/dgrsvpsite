Django Girls Workshop

During each of our workshops, 30-60 women build their first web application using HTML, CSS, Python and Django using the Django Girls Tutorial.

Code Camp I (Beginner)

A 2-3 month code camp to dive into the basics that we were introduced to in the first workshop.

Code Camp II(Advanced)

A 2-3 month project based code camp where attendees will impliment the knowledge from the previous code camp to creat their personal project.
